# **Maritime Ship Emissions Prediction**

## **Overview**

This project focuses on developing a robust predictive model to estimate carbon emissions and fuel consumption of maritime vessels based on their operational and environmental data. By accurately forecasting these metrics, the system provides actionable insights for optimizing fuel efficiency and ensuring environmental compliance in the maritime industry.

## **Key Features**

* **End-to-End ML Pipeline**: Implementation of the full Machine Learning lifecycle, from raw data preprocessing to model evaluation.
* **Advanced Regression Modeling**: Utilizing state-of-the-art gradient boosting algorithms for high-precision predictions.
* **Time-Series Analysis**: Handling temporal operational data to capture long-term trends and patterns.
* **Feature Engineering**: Extracting meaningful features from environmental and operational variables to improve model performance.

## **Tech Stack**

* **Language**: Python.
* **Data Processing**: Pandas, NumPy.
* **Machine Learning**: Scikit-learn, XGBoost, LightGBM.
* 
**Visualization**: Matplotlib, Seaborn.


## **Methodology**

1. **Data Preprocessing**: Handled missing values, outliers, and performed data normalization to ensure consistency.
2. **Feature Engineering**: Developed new operational indicators and transformed time-series data into supervised learning format.
3. **Model Selection**: Experimented with various algorithms, including Random Forest, XGBoost, and LightGBM.
4. **Hyperparameter Tuning**: Optimized model parameters to minimize error metrics such as MAE and RMSE.
5. **Evaluation**: Assessed model performance using $R^2$ score and other statistical indicators.

## **Results**

* Achieved an **$R^2$ score of 0.92** on the test set, demonstrating high reliability in fuel consumption forecasting.
* Successfully identified key factors influencing carbon emissions, aiding in strategic decision-making for greener maritime operations.
